# Página de checkout de viagem

Este projeto consiste em uma página de checkout de viagem que permite aos usuários escolherem datas de viagem, quantidade de passageiros e etc.

## Instruções de Execução Local

### Pré-requisitos
- Node.js instalado
- NPM ou Yarn

### Instalação

1. Clone o repositório: `git clone (https://github.com/EduardaDT/Pagina-de-checkout-de-viagem)`
2. Entre no diretório do projeto: `cd Pagina-de-checkout-de-viagem`
3. Instale as dependências: `npm install` ou `yarn install`

### Execução

Para executar o projeto localmente, utilize o seguinte comando: `npm start`
Isso iniciará o servidor local. Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.
